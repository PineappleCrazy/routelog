Requirements

  • Python 3.x installed on your system
  • Windows operating system (for batch file execution)
  • Basic understanding of airport ICAO codes

Installation Steps

  1. Download the repository files
  2. Run _routelog.bat to launch the program
  3. Select your aircraft types using function 9
  4. Import official routes through Data Settings (function 13)
  5. Set your home FIR using function 10
